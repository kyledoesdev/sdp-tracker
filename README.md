## Sleep Deproved Podcast Tracker

> **This project has been archived and is no longer actively maintained.**

The Sleep Deprived Podcast (SDP) was a call-in comedy podcast where hosts and listeners discussed a wide range of topics in an open, unscripted format. The show built a dedicated community over the course of four seasons before wrapping up in January 2026. SDP Tracker was a companion web application built to catalog and track episode data, call-in segments, and discussion topics across all seasons of the podcast. Users could browse episodes, view guest appearances, and explore stats about the show's history. The app was built with Laravel, Livewire, and Flux UI. It served as both a fan resource and a personal project to experiment with full-stack Laravel development. With the podcast concluded, the application has been taken offline and this repository has been archived for reference.
